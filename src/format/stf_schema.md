# STF File Schema
Schema index: <https://schema.stfform.at>

* **File Schema** [`https://schema.stfform.at/stf.ksy`](https://schema.stfform.at/stf.ksy)

	The STF file schema is specified using [Kaitai Struct](https://kaitai.io/).

* **Json Definition Schema** [`https://schema.stfform.at/stf.schema.json`](https://schema.stfform.at/stf.schema.json)

	Part of the STF binary file is the JSON definition. Its schema is defined using [JSON Schema](https://json-schema.org).
